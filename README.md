# Industrial-Electricity-Calculators

this library contains Industrial Electricity related calculators.

it is implemented based on [CalculatorEngine](https://github.com/araxis/CalculatorEngine).

## Calculators

*   Active Power ( x4 )
*   Apparent Power ( x3 )
*   Reactive Power ( x2 )
*   Current ( x2 )
*   Voltage ( x3)
